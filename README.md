# Sharma Coaching Classes — Client Demo Website

A professional business website built for a local coaching institute 
in Badlapur, Maharashtra using AI-assisted development tools.

## 🔗 Live Demo
**[View Live Website](https://v0-sharma-coaching-website.vercel.app)**

## 📌 About This Project
This website was built as part of a freelance web development initiative 
targeting local businesses in Badlapur that have no online presence.

The client was approached via WhatsApp outreach, and this demo was 
built and delivered within hours of the first conversation.

## ✨ Features
- Hero section with tagline and CTA buttons
- Stats section (500+ students, 15+ years experience, 95% success rate)
- About section with mission and values
- Courses offered (Science, Maths, English — Classes 8–12)
- Testimonials section
- Contact section with location details
- Fully responsive design

## 🛠️ Built With
- v0.dev (AI-assisted UI generation)
- React / Next.js
- Tailwind CSS
- Vercel (deployment)
